# Support

This repo is maintained on a best-effort basis. Please use the channels
below rather than opening low-priority issues.

## Questions and ideas

Use [Discussions](https://github.com/kaelys-js/kaelys-js/discussions)
for questions, ideas, and show-and-tell.

## Bug reports and feature requests

Open a bug or feature request on the
[Issues](https://github.com/kaelys-js/kaelys-js/issues)
tab. Fill out the template; blank issues are disabled by policy.

## Security

Report security issues privately per [SECURITY.md](SECURITY.md) — never
via a public issue.
